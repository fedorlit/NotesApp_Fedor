✨ Características
•
Deep Dark Theme: Interfaz en gris oscuro para descanso visual.
•
Persistencia: Almacenamiento local seguro con Room.
•
Orden Inteligente: Alterna entre fecha (recientes) y título (A-Z).
•
UX Adaptativa: Soporte para Edge-to-Edge y desplazamiento del botón con el teclado.
🛠️ Tech Stack
•
Kotlin + Compose (UI)
•
Room (DB) + Flow (Reactive)
•
MVVM + Navigation Compose
