# 📝 Notes App
Aplicación de notas minimalista construida con **Jetpack Compose** y **Room**.

## ✨ Características
* **Deep Dark Theme:** Interfaz en gris oscuro para descanso visual (evitando el negro puro).
* **Persistencia:** Almacenamiento local robusto con Room Database.
* **Orden Inteligente:** Alterna rápidamente entre fecha (más recientes) y título (A-Z).
* **UX Adaptativa:** Soporte nativo para *Edge-to-Edge* y botón flotante que reacciona al teclado.

## 🛠️ Tech Stack
* **Kotlin** + **Jetpack Compose** (UI declarativa)
* **Room** (Base de datos local) + **Flow** (Flujos de datos reactivos)
* **MVVM Architecture** + **Navigation Compose**

## 🚀 Inicio Rápido
