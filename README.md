## 🛠️ Tecnologías utilizadas
* **Kotlin** + **Jetpack Compose** (UI declarativa)
* **Room** (Base de datos local) + **Flow** (Flujos de datos reactivos)
* **MVVM Architecture** + **Navigation Compose**
